# Python Game

I made a simple game where the player operates a running girl who has to collect little soot sprites that appear in various locations across the screen. I implemented some custom music that loops while the game is running and sound effects when each sprite is collected. I also made three custom levels that get increasingly difficult, which the game progress through. When the last level is completed, the game ends and a "You win" screen is displayed.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Download the repo and required software
2. Press play in top right of the helloworld.py to build the project

Instructions for using the software:

1. Click play button in top right to launch the game
2. Use the arrow keys on the keyboard to move your player and collect sprites
3. Clear all the sprites in all the levels to win the game!
